- if文
    ```
    if (条件式1) {
        処理1
    }
    else if (条件式2) {
        処理2
    }
    else {
        処理3
    }
    ```
  - 処理の処理の部分が1文しかない場合は、`{ }`を省略可能
    ```
    if (x < 10)
        cout << "xは10より小さい" << endl;
    ```
  - 
- 比較演算子
  - xとyは等しい
    - `x == y`
  - xとyは等しくない
    - `x != y`
  - xはyより大きい
    - `x > y`
  - xはyより小さい
    - `x < y`
  - xはy以上
    - `x >= y`
  - xはy以下
    - `x <= y`
- 論理演算子
  - `!(条件式)
    - 条件式の結果の反転
  - `条件式1 && 条件式2`
    - 条件式1が真かつ条件式2が真
  - `条件式1 || 条件式2`
    - 条件式1が真または条件式2が真
